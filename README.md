  ## 💼 Work @ Synthlinx (proprietary — not open-sourced)

  **🔒 AIM — LLM Data-Migration Platform** · multi-tenant · financial
  - LLM-powered platform that maps schemas between banking core systems, auto-approving high-confidence mappings and routing the rest to human review.
  - Confidence engine blends a learning store, model confidence, vector similarity, and data-profile signals to gate decisions on zero-error-tolerance financial fields.
  - RAG learning store on pgvector reuses human-verified mappings as few-shot context, so mapping accuracy improves with each migration.
  - PydanticAI + LiteLLM agents extract business rules from documents into sandboxed Python; runs multi-tenant on Kubernetes with Temporal-orchestrated, resumable pipelines.

  **🔒 Logistics — Multi-Agent Supply-Chain Optimization**
  - LangGraph supervisor routes natural-language questions to five specialist agents via function calling, passing extracted parameters into each agent.
  - Optimization agents run order consolidation (best-fit-decreasing bin-packing), fixed-day scheduling, pallet stacking, and drop-point centralization.
  - Generates and safely executes pandas/Plotly code at request time to produce insights, cost-saving recommendations, and CO₂ analysis with interactive charts.
  - Multi-turn conversation memory via MongoDB checkpointing; charts served from Azure Blob Storage to a React frontend.

  **🔒 Agentic-RAG Document-Chat Platform** · multi-LLM
  - Agentic RAG pipeline: rewrites follow-up questions, retrieves from a FAISS vector index, reranks, and answers with inline source citations.
  - Unified interface across 5 LLM providers (GPT, Claude, Gemini, Mixtral, Mistral) with per-request switching and automatic fallback.
  - Multi-format ingestion (PDF, Office, images, audio, URLs) with OCR and speech-to-text, plus summarization and auto question-generation.
  - LLM safety guardrails (prompt-injection detection, input/output sanitization) on JWT-secured FastAPI endpoints.
